Quokka(vscode extension) shortcut 

 *   - Press `Ctrl Shift P ` to create a new JavaScript File
 *   - Press `Ctrl K, J` to create a new JavaScript File
 *   - Press `Ctrl K, T` to create a new TypeScript File
 *   - Press `Ctrl K, L` to open an interactive sample from:
 *     https://github.com/wallabyjs/interactive-examples
 *
 * To start/restart Quokka on an existing file:
 *   - Press `Ctrl K, Q`	(IMPORTANT) 	

  * if Quokka show error like  "Cannot find module 'jsdom' Require stack: - .\index.js" then copy below command and run it in terminal

cd ~/.quokka/
npm i jsdom-quokka-plugin jsdom
